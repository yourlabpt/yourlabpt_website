## 🚀 Get Started in 2 Minutes

### Option 1: Test Without Backend
1. Install VS Code extension "Live Server"
2. Right-click `index.html` → "Open with Live Server"
3. ✅ Done! Website opens

### Option 2: Full Setup with Backend
**Mac/Linux:**
```bash
cd website/server
./start.sh
```

**Windows:**
```bash
cd website/server
start.bat
```

**Then visit:** http://localhost:3000

## 💾 Data Storage

Your inquiries are saved in two places:

1. **Browser Local Storage**
   - View in console: `showSavedConversations()`
   - Persists across sessions
   - No backend needed

2. **Server Storage** (if using backend)
   - Files in `server/inquiries/`
   - Permanent storage
   - Easy to backup and export
