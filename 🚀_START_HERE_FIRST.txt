METHOD 1️⃣ - FASTEST WAY (2 minutes, no backend setup)
──────────────────────────────────────────────────────────────────────────────

This is the quickest way to test your website locally.

STEPS:
  1. Open VS Code
  2. Open the Extensions panel (Ctrl+Shift+X / Cmd+Shift+X)
  3. Search for "Live Server" by Ritwick Dey
  4. Click Install
  5. In the file explorer, right-click on "index.html"
  6. Select "Open with Live Server"
  7. Your website will open in the browser! 🎉

That's it! Your website is now running locally.

TO VIEW SAVED DATA:
  • Open browser Developer Console (F12 or right-click → Inspect → Console)

  • Press Enter

METHOD 2️⃣ - FULL SETUP (5 minutes, with backend & admin dashboard)
──────────────────────────────────────────────────────────────────────────────

This method sets up the complete system with a backend server and admin 
dashboard to view and manage inquiries.

STEPS:

For Mac/Linux users:
  1. Open Terminal
  2. Navigate to: cd website/server
  3. Run: ./start.sh
  4. The script will install dependencies and start the server
  5. You'll see: "🚀 YourLab Chat API running on http://localhost:3000"

For Windows users:
  1. Open Command Prompt (cmd) or PowerShell
  2. Navigate to: cd website/server
  3. Run: start.bat
  4. The script will install dependencies and start the server
  5. You'll see: "🚀 YourLab Chat API running on http://localhost:3000"

For manual setup (any OS):
  1. Open Terminal/Command Prompt
  2. Navigate to: cd website/server
  3. Run: npm install
  4. Run: npm start
  5. You'll see: "🚀 YourLab Chat API running on http://localhost:3000"

THEN:
  • Open browser and go to: http://localhost:3000
  • Your website is now live! 🎉
  • To view admin dashboard: http://localhost:3000/admin.html
  • To see all saved inquiries: http://localhost:3000/admin.html

TO STOP THE SERVER:
  • Press Ctrl+C in the terminal

🔧 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Website won't load?
  • Make sure you're using the correct URL (http://localhost:3000)
  • Make sure the backend is running (if using METHOD 2️⃣)
  • Check browser console for errors (F12)

Data not saving?
  • Make sure you complete the conversation (provide email & phone)
  • Check if localStorage is enabled (F12 → Application → Local Storage)
  • Check browser console for errors

Backend won't start?
  • Make sure you have Node.js installed (download from nodejs.org)
  • Try deleting node_modules folder and running npm install again
  • Make sure port 3000 is not already in use

Live Server not working?
  • Make sure the Live Server extension is installed
  • Try reloading the page
  • Try stopping and starting Live Server again
